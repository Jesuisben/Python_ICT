### **3. 모델 정의 (movies/models.py)**
SQLite 테이블에 맞게 모델을 생성합니다.


from django.db import models

class Movie(models.Model):
    movieCd = models.CharField(max_length=10, primary_key=True)
    movieNm = models.CharField(max_length=255)
    movieNmEn = models.CharField(max_length=255, blank=True, null=True)
    prdtYear = models.IntegerField()
    openDt = models.CharField(max_length=10, blank=True, null=True)
    typeNm = models.CharField(max_length=50)
    prdtStatNm = models.CharField(max_length=50)
    nationAlt = models.CharField(max_length=255)
    genreAlt = models.CharField(max_length=255)
    repNationNm = models.CharField(max_length=50)
    repGenreNm = models.CharField(max_length=50)

    def __str__(self):
        return self.movieNm
# end class

### **6. CSV 데이터 가져오기 (movies/management/commands/load_movies.py)**
Django 명령어를 만들어 CSV 데이터를 데이터베이스에 삽입합니다.

```bash
mkdir -p movies/management/commands
touch movies/management/commands/__init__.py
touch movies/management/commands/load_movies.py
```

`movies/management/commands/load_movies.py` 파일을 다음과 같이 작성합니다.

```python
import csv
from django.core.management.base import BaseCommand
from movies.models import Movie

class Command(BaseCommand):
    help = "Load movies from kmdb_get_movie_list.csv into the database"

    def handle(self, *args, **kwargs):
        csv_file = "kmdb_get_movie_list.csv"

        with open(csv_file, newline="", encoding="utf-8") as file:
            reader = csv.reader(file)
            next(reader)  # 헤더 건너뛰기

            for row in reader:
                Movie.objects.update_or_create(
                    movieCd=row[0],
                    defaults={
                        "movieNm": row[1],
                        "movieNmEn": row[2],
                        "prdtYear": int(row[3]),
                        "openDt": row[4] if row[4] else None,
                        "typeNm": row[5],
                        "prdtStatNm": row[6],
                        "nationAlt": row[7],
                        "genreAlt": row[8],
                        "repNationNm": row[9],
                        "repGenreNm": row[10],
                    }
                )

        self.stdout.write(self.style.SUCCESS("CSV 데이터 로드 완료!"))
```

이제 다음 명령어를 실행하면 CSV 데이터를 데이터베이스에 삽입할 수 있습니다.

```bash
python manage.py load_movies
```

### **7. 영화 리스트 API 생성 (movies/views.py)**
Django의 `JsonResponse`를 사용하여 REST API를 간단히 구현합니다.

```python
from django.http import JsonResponse
from .models import Movie

def movie_list(request):
    movies = Movie.objects.all().values()
    return JsonResponse(list(movies), safe=False)
```

### **8. URL 설정 (movies/urls.py)**
